42 C++ Modules (cpp00 - cpp09)

This repository contains my C++ projects from the 42 curriculum. Each project is in its own folder and explores different C++ concepts, from basic syntax to advanced object-oriented programming, templates, and STL usage.

Projects Overview
cpp00 – Basic C++

Introduces basic C++ syntax, variables, functions, and simple classes. Focuses on understanding compilation, execution, and basic class design.

cpp01 – Constructors & Operators

Covers constructors, copy constructors, destructors, assignment operators, and operator overloading. Teaches how to manage object copies and assignments correctly.

cpp02 – Inheritance & Polymorphism

Introduces single inheritance, virtual functions, and polymorphism. Demonstrates how derived classes can override base class behavior.

cpp03 – Encapsulation & Class Design

Focuses on proper class design, encapsulation, access modifiers, and getters/setters. Teaches best practices for managing object data.

cpp04 – Advanced Inheritance

Covers multiple inheritance, virtual inheritance, and object slicing. Explains how to handle complex inheritance hierarchies safely.

cpp05 – Templates & Exceptions

Introduces function and class templates, along with exception handling. Teaches generic programming and error management.

cpp06 – Advanced Templates & Dynamic Memory

Covers advanced template usage, dynamic memory allocation, and smart pointers. Focuses on writing reusable and memory-safe code.

cpp07 – Standard Library Containers

Explores STL containers like vector, map, and set. Covers iterators, algorithms, and container management in C++.

cpp08 – Polymorphic Containers & RTTI

Focuses on storing heterogeneous objects, runtime type identification (RTTI), and safe type casting. Explains polymorphism in container contexts.

cpp09 – Final Module

A comprehensive module combining all previously learned concepts into larger exercises. Includes complex class design, inheritance, templates, and STL usage.
